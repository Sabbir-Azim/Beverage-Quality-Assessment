# Veverage-Quality-Assessment

# How to run?
### STEPS:

1. Create a Vitual Enviroment

```bash
conda create -n beverage python=3.8 -y
```


2. Activate the enviroment

```bash
conda activate beverage
```


3. Install Requirements

```bash
pip install -r requirements.txt
```


4. Execute app


```bash
python app.py
```
